Projects

Utilities Spend Tracker

Utilities Spend Tracker is a full-stack web application to help users manage and track their monthly utility expenses (Water, Electricity, Gas, Internet) using React for the frontend, Spring Boot for the backend, and MySQL for data storage. The app features Apache Tika for PDF bill uploads and parsing, allowing users to easily extract and categorize their bill information. I'm focused on developing dynamic visualizations that showcase the data in an insightful way, helping users better understand their spending trends.

Night Out

Night Out is designed to be an interactive app that allows the user to plan their “night out”. The main function of the app is tracking user location to display bars near them, as well as, display the statistics of a selected bar. The app includes interactive features like a scrollable map, a drop down menu with bars, and clickable markers on their respective bar locations.
